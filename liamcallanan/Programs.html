Mine(Craft) Sweeper

				

				
Mine(Craft) Sweeper is a program that I coded based on the games MineCraft and Minsweeper. I made it with the help of my Senior Project mentor, Ben LeVeque, who gave me this project so that I could learn the basics of Javascript and HTML. Doing this gave me some basic knowledge to work with, allowing me to understand enough to make this website.
